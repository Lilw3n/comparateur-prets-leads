# Instructions pour ajouter les images des équipements

## Images à ajouter

Vous devez ajouter deux images dans le dossier `frontend/public/images/` :

1. **Matterport Pro 2** : `matterport-pro2.png`
   - Format recommandé : PNG avec fond transparent ou JPG
   - Taille recommandée : 400x400px minimum
   - Sources possibles :
     - Site officiel Matterport : https://matterport.com/pro2
     - Recherche Google Images : "Matterport Pro 2 camera"
     - Photos de votre propre équipement

2. **Fujifilm X-T2** : `fujifilm-xt2.png`
   - Format recommandé : PNG avec fond transparent ou JPG
   - Taille recommandée : 400x400px minimum
   - Sources possibles :
     - Site officiel Fujifilm : https://www.fujifilm-x.com/
     - Recherche Google Images : "Fujifilm X-T2 camera"
     - Photos de votre propre équipement

## Emplacement des fichiers

```
frontend/
  public/
    images/
      matterport-pro2.png
      fujifilm-xt2.png
```

## Où les images apparaîtront

Les images seront affichées dans la page **Présentation** (`/presentation`) dans la section "Visites Virtuelles 360°", juste avant l'exemple de visite virtuelle.

## Note

Si les images ne sont pas trouvées, un texte de remplacement s'affichera automatiquement avec le nom de l'équipement.
