# Instructions d'activation du thème enfant

## Étapes pour activer le thème enfant

1. **Connectez-vous à l'administration WordPress**
   - Allez sur `https://lilwen54.fr/wp-admin`
   - Connectez-vous avec vos identifiants

2. **Accédez à la section Thèmes**
   - Dans le menu de gauche, cliquez sur **Apparence** > **Thèmes**

3. **Activez le thème enfant**
   - Recherchez le thème **"Lilwen54 Child"**
   - Cliquez sur **Activer**

4. **Vérification**
   - Le thème enfant devrait maintenant être actif
   - Votre site devrait continuer à fonctionner normalement
   - Les styles du thème parent sont automatiquement chargés

## Avantages immédiats

✅ **Sécurité renforcée** : Vos modifications ne seront plus perdues lors des mises à jour  
✅ **Personnalisation sécurisée** : Vous pouvez modifier les styles dans `style.css`  
✅ **Fonctionnalités extensibles** : Ajoutez du code dans `functions.php` sans risque  

## Prochaines étapes

1. **Personnaliser les styles** : Modifiez `wp-content/themes/lilwen54-child/style.css`
2. **Ajouter des fonctionnalités** : Utilisez `wp-content/themes/lilwen54-child/functions.php`
3. **Créer des templates** : Copiez les templates du thème parent si nécessaire

## Important

⚠️ **Ne modifiez jamais directement le thème parent** (twentytwentyfive)  
✅ **Toutes vos modifications doivent être dans le thème enfant**
