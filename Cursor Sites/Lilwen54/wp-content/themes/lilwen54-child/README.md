# Thème Enfant Lilwen54

## Description

Ce thème enfant est basé sur **Twenty Twenty-Five** et permet de personnaliser le site Lilwen54 sans perdre les modifications lors des mises à jour du thème parent.

## Avantages d'un thème enfant

✅ **Sécurité** : Les modifications ne sont pas perdues lors des mises à jour WordPress  
✅ **Personnalisation** : Vous pouvez modifier les styles et fonctionnalités sans toucher au thème parent  
✅ **Maintenance** : Le thème parent reste intact et peut être mis à jour en toute sécurité  
✅ **Performance** : Pas d'impact sur les performances du site  

## Structure des fichiers

```
lilwen54-child/
├── style.css          # Styles personnalisés
├── functions.php       # Fonctions et hooks personnalisés
└── README.md          # Ce fichier
```

## Utilisation

1. **Activer le thème** : Allez dans Apparence > Thèmes et activez "Lilwen54 Child"

2. **Personnaliser les styles** : Modifiez le fichier `style.css` pour ajouter vos styles personnalisés

3. **Ajouter des fonctionnalités** : Utilisez `functions.php` pour ajouter des hooks, filtres ou fonctions personnalisées

4. **Créer des templates** : Vous pouvez créer des templates personnalisés en copiant ceux du thème parent dans ce dossier

## Bonnes pratiques

- ✅ Toujours tester les modifications sur un site de développement
- ✅ Utiliser un système de contrôle de version (Git)
- ✅ Documenter vos modifications
- ✅ Faire des sauvegardes régulières
